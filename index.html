What is Diaspora?

Diaspora means diasporic in spanish and as per in the name it means ‘being, or relating to any group that has been dispersed outside its traditional homeland, either involuntarily or by migration’ This concept is applied to a programming exercise that has as main purpose to create relationships between the followers of a social media network, github in this scenario, and to track in a graphic way the connection between them.
